AwesomeSearch

Awesome-Search

Makes you find what you want in awesome lists more quickly.

Why

In a nutshell, there are many awesome lists in the world :)
We hope to build an application to access them more quickly.
That's why awesome search was born.

Features

Access and search every awesome repo collected in sindresorhus/awesome in one page without pain.
Access an awesome repo by https://awesomelists.top/#repos/sindresorhus/awesome-nodejs.(you can bookmark your favorite!)
We run schedule workers in the background to parse the links in every awesome list, so it's up-to-date :)
Categories of an awesome repo is supported if the that repo uses headings.
Use Github API to retrieve README file of an awesome repo, so it's up-to-date.
Search links in a specified repo(support most of repos) and repos listed in sindresorhus/awesome.
